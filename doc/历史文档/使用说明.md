# X传输工具 — 使用说明

面向会议与办公局域网场景：在同一 Wi‑Fi 或同一有线网络下收发文件，并在对方同意时进行远程协助。

## 第一次使用

1. 打开 **设置**，填写 **昵称**（方便别人认出您）。
2. 按需填写 **分组标签**（如会场名、区域），便于列表里自动归类。
3. 选择 **接收文件保存位置**（一个您有权限写入的文件夹），点击 **保存并应用**。
4. 若 Windows 询问是否允许访问网络，请选择 **允许**。

## 发送文件

在首页 **其它用户** 中展开标签，找到对方，将文件 **拖到对方那一行** 上，按提示可选填一句说明，即可发送。

## 接收文件

保持本应用打开即可。收到的文件会保存在您设置的文件夹中，首页下方可查看记录。

## 远程协助（看别人或操作对方电脑）

- **被协助的一方（本机要被连）**  
  在 **设置** 中打开 **允许其他人远程控制本电脑**。打开后，本应用会在后台准备好远程协助；同一局域网内其他人可直接连接，**无需再输入密码**（与官方版本约定一致即可）。

- **协助别人的一方**  
  在列表里找到对方，点击行尾的 **显示器图标** 进入远程协助。若该图标为灰色，表示对方未开启远程协助。

请在取得对方同意后再进行远程连接。操作系统可能询问屏幕录制、辅助功能等权限，请按提示允许。

## 看不到其他电脑时

- 确认双方连接的是 **同一网络**（不要用访客 Wi‑Fi 或开启了「设备隔离」的路由器功能）。
- 确认对方也已打开本应用。
- 在 **连接问题说明** 中查看常见原因，或复制 **简要诊断信息** 给技术支持。

## 版本信息
